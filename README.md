# 🎓 Student Database Management System

A simple and interactive Student Database Management System built using **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript**.

This project helps manage student records efficiently by allowing users to add, view, update, search, and delete student information through a user-friendly interface.

---

## 🚀 Features

* ➕ Add New Students
* 👀 View Student Records
* ✏️ Edit Existing Details
* 🗑️ Delete Student Records
* 🔍 Search Students Instantly
* 📊 Total Student Counter
* 💾 MongoDB Database Integration
* 🎨 Responsive and Modern UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/suhanipandey225-cyber/Student-Database-Management-System.git
```

### Open Project Folder

```bash
cd Student-Database-Management-System
```

### Install Dependencies

```bash
npm install
```

### Start MongoDB

Make sure MongoDB is running on your system.

### Run Project

```bash
npm run dev
```

or

```bash
node server.js
```

### Open Browser

```txt
http://localhost:3000
```
